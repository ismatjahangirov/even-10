Melumat sehivesi

Frontend texnologiyaları müasir veb saytların yaradılmasında əsas rol oynayır. HTML, CSS və JavaScript birlikdə veb səhifələrin strukturunu, dizaynını və funksionallığını təmin edir.


Html hakkinda melumat

  • HTML veb səhifələrin quruluşunu yaratmaq üçün istifadə olunur.
  • HTML vasitəsilə başlıqlar, mətnlər və şəkillər səhifəyə əlavə edilir.
  • HTML internetdə saytların əsasını təşkil edən dildir.
  • HTML elementlər və taglar vasitəsilə işləyir.


CSS hakkinda melumat
  • CSS veb səhifələrin dizaynını və görünüşünü dəyişmək üçünCSS veb səhifələrin dizaynını və görünüşünü dəyişmək üçün
  • CSS vasitəsilə rənglər, şriftlər və ölçülər tənzimlənə bilir.
  • CSS istifadə edərək veb saytlar daha cəlbedici görünür.


JavaScript hakkinda melumat
  • JavaScript web səhifələrə interaktivlik əlavə edir
  • Düymələr, animasiyalar və dinamik dəyişikliklər JavaScript ilə edilir
  • Bu texnologiya istifadəçi ilə sayt arasında qarşılıqlı əlaqə yaradır
